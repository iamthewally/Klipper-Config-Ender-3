Extruder:

- E3D Hemera + Volcano + Tungsten nozzle for running CF filaments

Biqu Crab Mounting System

3DFused X rail

Dyze Horizon ABL (Mounted between Crab and Carriage

CHeap LED light from amazon taped to top rail
